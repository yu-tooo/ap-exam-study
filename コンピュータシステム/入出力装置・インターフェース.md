# 入出力装置・インターフェース

## 入力装置

### キーボードの性能指標

- **Nキーロールオーバー**: 同時に押された複数のキーを正しく認識できる**最大同時押下数**。Nが大きいほどゲーム用途などで有利。

## 出力装置

### ディスプレイの性能指標

- **解像度**: ディスプレイ全体のピクセル数（例: 1920×1080）。同じ解像度でも画面サイズが大きくなると粒度が粗くなる。
- **dpi**（dots per inch）: 1インチあたりのピクセル数。高いほど細かい表示が可能。

### プリンタの性能指標

| 単位 | 意味 |
|------|------|
| **cps**（characters per second） | 1秒あたりの印刷文字数（シリアルプリンタ）。 |
| **ppm**（pages per minute） | 1分あたりの印刷ページ数（ページプリンタ）。 |
| **dpi**（dots per inch） | 1インチあたりのドット数。印刷の解像度を表す。 |

## 入出力インターフェース

### シリアルインターフェース

**1本の信号線**で**1ビットずつ**データを送る方式。配線が単純で長距離伝送に向き、現在の主流。

#### 有線シリアル

| 規格 | 特徴 |
|------|------|
| **USB**（Universal Serial Bus） | 汎用的なシリアル接続規格。電源供給も可能。バージョンにより速度が大きく異なる。 |
| **HDMI**（High-Definition Multimedia Interface） | 映像と音声を1本のケーブルで伝送するデジタルAV規格。 |
| **DisplayPort** | 映像伝送規格。**デイジーチェーン**（ディスプレイを鎖状に数珠つなぎする接続）が可能。 |

### パラレルインターフェース

**複数の信号線**で**複数ビットを同時に**送る方式。短距離では高速だが、配線間の干渉や同期ずれが発生しやすく、長距離・高速化には不利。現在は限定的な用途のみで使われる。

### 無線インターフェース

| 規格 | 特徴 |
|------|------|
| **Bluetooth** | 2.4GHz帯を使用。**周波数ホッピング**により干渉しにくい。 |
| **BLE**（Bluetooth Low Energy） | 低消費電力版Bluetooth。**ボタン電池で数年動作**するため、IoT機器やウェアラブルデバイスで利用。 |
| **IrDA**（Infrared Data Association） | 赤外線を利用した近距離通信。**障害物に弱い**ためあまり使われていない。 |
| **RFID** / **NFC** | 数センチの近距離通信。**電子マネー**や交通系ICカードで利用。 |

## 補助記憶装置（ハードディスク）

### 構造の用語

ハードディスクは複数の円盤（プラッタ）が積層された構造になっている。

```
[側面図]                    [上から見たプラッタ]

┌─────────────┐                  外周
│ プラッタ1     │  ←── トラック       ↓
├─────────────┤   ┌───┬───┬───┐
│ プラッタ2     │   │   │   │   │ ← セクタ
├─────────────┤   ├───┼───┼───┤
│ プラッタ3     │   │ トラック  │
└─────────────┘   └───┴───┴───┘
       ↑                ↑
       └─ 同心円が同じ位置にある
          複数プラッタのトラック群
          ＝ シリンダ
```

| 用語 | 意味 |
|------|------|
| **プラッタ** | 磁気記録用の円盤。1台のディスクに複数枚積層されている。 |
| **トラック** | プラッタ上の同心円状の記録領域。 |
| **シリンダ** | 複数プラッタの**同じ位置にあるトラック**の集合。ヘッドを動かさずにアクセスできる単位。 |
| **セクタ** | トラックを放射状に区切った最小の記録単位。**1セクタ＝512バイト**が伝統的。 |
| **クラスタ** | 複数セクタの集まり。OSがファイルを管理する単位。 |

主記憶装置との間の読み書きは**クラスタ単位**で行われる。

### アクセス時間

```
アクセス時間 = サーチ時間 + シーク時間 + 転送時間
```

| 用語 | 説明 |
|------|------|
| **シーク時間**（位置決め時間） | 磁気ヘッドを目的のトラック（シリンダ）まで移動させる時間。 |
| **サーチ時間**（回転待ち時間） | プラッタの回転により、目的のセクタがヘッドの下に来るまで待つ時間。平均的には**1回転の半分**の時間。 |
| **転送時間** | 目的のセクタからデータを読み書きしてメモリに転送する時間。 |
