# システム戦略

## 業務フェーズと支援システム

生産・物流・販売の各フェーズで活用される代表的なシステム。

| フェーズ | 必要な情報・システム |
|---|---|
| **生産** | 生産計画（**MRP**）、在庫情報（**SCM**）、**ERP** |
| **物流** | 在庫情報（**SCM**）、顧客情報（**CRM**）、**ERP** |
| **販売** | 在庫情報（**SCM**）、顧客情報（**CRM**）、**ERP** |

### ERP（Enterprise Resource Planning）

経営に必要な資源（ヒト・モノ・カネ・情報）を有効活用するための統合的な計画・管理。基幹業務（会計・人事・販売・生産など）を統合した**ERPパッケージ**を導入して実現することが多い。

## 業務改善・改革（BPR と BPM）

業務プロセスを見直す代表的な2つのアプローチ。

| 観点 | **BPR**（Business Process Reengineering） | **BPM**（Business Process Management） |
|---|---|---|
| **適用範囲** | 全社的に抜本的な改革 | 既存業務を再構築し、業務を改善 |
| **改善の視点** | トップダウン方式 | ボトムアップ方式 |
| **実施の回数・期間** | 基本的に一度きり | 繰り返し継続し、長期的に改善 |

### BPR・BPMとは

- **BPR**: 既存の業務プロセスを根本的に見直し、コスト・品質・スピードなどを劇的に向上させる改革手法。組織や情報システムも含めて再設計する。
- **BPM**: 業務プロセスを継続的にモニタリングし、PDCAを回して改善し続ける管理手法。BPRが「一度きりの改革」であるのに対し、BPMは「継続的な改善」を目指す。

## サプライチェーンマネジメント（SCM）

原材料の調達から製品が消費者に届くまでの**サプライ（供給）**の流れを最適化するため、関係企業間の情報共有と業務管理をシステムで統合する仕組み。

### EDI（Electronic Data Interchange）

企業間で商取引に関するデータを電子的に交換しやすくするために、**標準化されたメッセージ形式**でやり取りする仕組み。

## マーケティング・営業支援ツール

顧客との関係構築の段階に応じて使い分けられる。

```
1. リード獲得
       ↓
2. リードの育成 ── MAツール（継続的なコミュニケーションを自動化、見込み客との関係強化）
       ↓
3. 商談 ──────── SFA（営業活動のデータベース化で営業効率を向上）
       ↓
4. クロージング
       ↓
5. 受注 ──────── CRM（顧客情報のデータベース化で顧客生涯価値（LTV）を上げる）
       ↓
6. アップセル
```

| 略称 | 正式名称 | 役割 |
|---|---|---|
| **MA** | Marketing Automation | マーケティング活動の自動化 |
| **SFA** | Sales Force Automation | 営業支援システム |
| **CRM** | Customer Relationship Management | 顧客関係管理 |

## HRテック

ITを活用して人事管理を高度化する仕組み（採用、人事評価、労務、タレントマネジメントなど）。

## CGM・UGC・CMS・API

| 略称 | 正式名称 | 説明 |
|---|---|---|
| **CGM** | Consumer Generated Media | 一般ユーザーがコンテンツを作成するメディアプラットフォーム（口コミサイト、SNSなど） |
| **UGC** | User Generated Content | 個人ユーザーが作成したコンテンツそのもの |
| **CMS** | Content Management System | 専門知識がなくてもWebサイトを作成・更新できるシステム（WordPressなど） |
| **API** | Application Programming Interface | アプリケーションの機能を別のアプリから呼び出す仕組み |

## クラウドサービス

利用者が管理する範囲の違いによる分類。

| 形態 | 利用者が管理する範囲 | 提供される範囲 |
|---|---|---|
| **オンプレミス** | すべて自社で管理（クラウド未使用） | — |
| **IaaS** | OS・ミドルウェア・アプリ・データ | サーバ、ストレージ、ネットワーク |
| **PaaS** | アプリ・データ | IaaSの範囲＋OS、ミドルウェア |
| **SaaS** | データの一部のみ | PaaSの範囲＋アプリケーション |

## 仮想通貨

**ブロックチェーン**（分散型台帳技術）をもとに開発されたデジタル通貨。中央管理者を持たず、取引履歴を分散ネットワーク上で改ざんしにくい形で記録する。
