# 会計・財務

**シラバス大分類：22. 企業活動**

## 損益計算書（PL）の利益構造

**損益計算書**（Profit and Loss statement, PL）は、一定期間の収益と費用から利益を計算した財務諸表。

### 収益と費用の内訳

- **収益** = 売上高 ＋ 営業外収益（受取利息など）＋ 特別利益（保険金収入など）
- **費用** = 売上原価 ＋ 販売費・一般管理費（電気代など）＋ 営業外費用 ＋ 特別損失（台風による設備破損など）
- **利益** = 収益 − 費用

### 段階的な利益計算

損益計算書では、本業の儲け→経常的な儲け→最終的な儲けと段階的に利益を表示する。

| 段階 | 計算式 | 意味 |
|---|---|---|
| **売上総利益**（粗利） | 売上高 − 売上原価 | 商品の販売そのものから得た利益 |
| **営業利益** | 売上総利益 − 販売費・一般管理費 | 本業から得た利益 |
| **経常利益** | 営業利益 ＋ 営業外収益 − 営業外費用 | 経常的な事業活動から得た利益 |
| **税引前当期純利益** | 経常利益 ＋ 特別利益 − 特別損失 | 臨時の損益を加味した利益 |
| **当期純利益** | 税引前当期純利益 − 法人税等 | 最終的に企業に残る利益 |

## 損益分岐点

**損益分岐点**（Break-Even Point, BEP）は、売上高と費用が等しくなり利益も損失も発生しない点。

- **変動費率** = 変動費 ÷ 売上高
- **固定費率** = 固定費 ÷ 売上高

### 損益分岐点の計算式

```
損益分岐点販売量 = 固定費 ÷ (販売単価 − 単位あたり変動費)
損益分岐点売上高 = 損益分岐点販売量 × 販売単価
              = 固定費 ÷ (1 − 変動費率)
```

### 計算例

固定費＝10万円、販売単価＝250円／個、変動費＝50円／個 のとき、

```
損益分岐点販売量 = 100,000 ÷ (250 − 50) = 500 個
損益分岐点売上高 = 500 × 250 = 125,000 円

別の解き方：
変動費率 = 50 ÷ 250 = 0.2
損益分岐点売上高 = 100,000 ÷ (1 − 0.2) = 125,000 円
```

## 財務諸表

企業の経営状況を報告する代表的な3つの計算書。

| 財務諸表 | 別名 | 表すもの | 期間軸 |
|---|---|---|---|
| **損益計算書**（PL） | Profit and Loss statement | 一定期間の経営成績（収益・費用・利益） | フロー（期間） |
| **貸借対照表**（BS） | Balance Sheet | 一時点の財政状態（資産・負債・純資産） | ストック（時点） |
| **キャッシュフロー計算書**（CF） | Cash Flow Statement | 一定期間の現金の流れ（増減と残高） | フロー（期間） |

### 貸借対照表

**貸借対照表**は、ある時点における資産・負債・純資産（資本）の状態を示す。

```
資本（純資産） = 資産 − 負債
```

#### 資産の内訳

- **流動資産**: 1年以内に現金化できる資産（現金、売掛金、棚卸資産など）
- **固定資産**: 現金化に1年以上かかる資産（建物、機械、土地など）

#### 負債の内訳

- **流動負債**: 1年以内に返済する負債（買掛金、短期借入金など）
- **固定負債**: 返済までに1年以上かかる負債（社債、長期借入金など）

### キャッシュフロー計算書

**キャッシュフロー計算書**は、一定期間における現金及び現金同等物の増減を示す。

1. **営業活動によるキャッシュフロー**: 本業による現金の出入り
2. **投資活動によるキャッシュフロー**: 設備投資や有価証券の売買による現金の出入り
3. **財務活動によるキャッシュフロー**: 借入や返済、増資などによる現金の出入り
4. **現金及び現金同等物の増減額**（1＋2＋3）
5. **現金及び現金同等物の期首残高**
6. **現金及び現金同等物の期末残高**（4＋5）

## 財務に関する指標

### 収益性・安全性・効率性の指標

| 指標 | 計算式 | 意味 |
|---|---|---|
| **自己資本比率** | 自己資本 ÷ 総資本 | 総資本のうち返済不要な資金の割合（高いほど財務健全性が高い） |
| **流動比率** | 流動資産 ÷ 流動負債 | 短期的な支払能力（一般に200%以上が望ましい） |
| **固定比率** | 固定資産 ÷ 自己資本 | 固定資産を自己資本でまかなえているか（低いほど安全） |
| **総資本回転率** | 売上高 ÷ 総資本 | 総資本をどれだけ効率的に売上に結び付けたか |
| **限界利益率** | 限界利益（売上高 − 変動費）÷ 売上高 | 売上が増えたときの利益増加の大きさ |
| **安全余裕率** | (売上高 − 損益分岐点売上高) ÷ 売上高 | 赤字になるまでの売上高の余裕 |

### 投資効率の指標

| 指標 | 計算式 | 意味 |
|---|---|---|
| **ROI**（Return On Investment） | 利益 ÷ 投下資本 | 投下した資本に対する利益の割合 |
| **ROA**（Return On Assets） | 当期純利益 ÷ 総資産 | 総資産をどれだけ効率的に利益に変えたか |
| **ROE**（Return On Equity） | 当期純利益 ÷ 自己資本 | 株主から預かった資本でどれだけ稼いだか |

## 減価償却

**減価償却**は、固定資産の取得費用を耐用年数にわたって分割して費用計上する会計処理。

| 方式 | 説明 | 特徴 |
|---|---|---|
| **定額法** | 毎年同額を償却する | 償却費が一定で計算が簡単 |
| **定率法** | 帳簿残高に一定率を掛けて償却する | 初年度ほど償却額が大きく、年々逓減する |

## 棚卸資産評価

期末在庫の評価額（および売上原価）を決める方法。

| 方式 | 考え方 | 例（仕入: 100円×3個 → 120円×2個、4個販売） |
|---|---|---|
| **先入先出法**（FIFO） | 先に仕入れたものから先に販売されたとみなす | 売上原価 = 100×3 + 120×1 = 420円<br>期末在庫 = 120×1 = 120円 |
| **移動平均法** | 仕入のたびに平均単価を計算し直す | 平均単価 = (100×3 + 120×2) ÷ 5 = 108円<br>売上原価 = 108×4 = 432円<br>期末在庫 = 108×1 = 108円 |

仕入単価が**上昇**する局面では、先入先出法のほうが売上原価が小さく利益が大きくなる。
