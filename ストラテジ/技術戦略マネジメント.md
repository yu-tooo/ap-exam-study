# 技術戦略マネジメント

## イノベータ理論

新しい商品やサービスが市場に普及していく流れを、消費者を**革新性**に基づいて5つの層に分類した理論（E. M. ロジャーズ提唱）。

| 順 | 層 | 割合（目安） | 特徴 |
|---|---|---|---|
| 1 | **イノベータ**（Innovators） | 約2.5% | 新しいものへの感度が高く、積極的に受け入れる |
| 2 | **アーリーアダプター**（Early Adopters） | 約13.5% | オピニオンリーダー。他の消費者層への火付け役となる |
| — | **── キャズム ──** | — | 普及の前に立ちはだかる深い溝 |
| 3 | **アーリーマジョリティ**（Early Majority） | 約34% | 商品が主流（マス）になるための鍵を握る |
| 4 | **レイトマジョリティ**（Late Majority） | 約34% | 周囲が使い始めてから採用する。市場の成熟期 |
| 5 | **ラガード**（Laggards） | 約16% | 製品が完全に普及してから、もしくは最後まで採用しない層 |

**キャズム**は、アーリーアダプターとアーリーマジョリティの間にある大きな溝。商品が主流市場に普及するためには、ここを越える必要がある（キャズム理論）。

## イノベーションの3つの障壁

研究から産業化（ヒット）に至るまでに立ちはだかる3つの段階的な壁。

```
1. 研究
   │
   │ ── 魔の川 ── （技術とニーズを結び付け、製品化に至る壁）
   │
2. 開発
   │
   │ ── 死の谷 ── （事業化に必要な資金・人材などのリソース不足の壁）
   │
3. 事業化
   │
   │ ── ダーウィンの海 ── （生産体制と流通の確保、市場で生き残る競争の壁）
   │
4. 産業化（ヒット）
```

| 障壁 | 場所 | 内容 |
|---|---|---|
| **魔の川** | 研究 → 開発 | 技術シーズを市場ニーズに結び付けて製品化できないと越えられない |
| **死の谷** | 開発 → 事業化 | 事業化に必要な資金・人材・経営資源が不足し、事業として立ち上げられない |
| **ダーウィンの海** | 事業化 → 産業化 | 生産体制・流通網の確立、競合との競争を勝ち抜き市場で生き残る必要がある |
