# 法務

**シラバス大分類：23. 法務**

## 知的財産権の全体像

知的財産権は大きく**著作権**と**産業財産権**に分かれる。

```
知的財産権
├── 著作権 ── 創作時に自動付与
│   ├── 著作者人格権（譲渡・相続不可）
│   └── 著作財産権（譲渡・相続可）
└── 産業財産権 ── 特許庁への届出が必要
    ├── 特許権
    ├── 実用新案権
    ├── 意匠権
    └── 商標権
```

## 著作権

著作物（思想・感情を創作的に表現したもの）の創作時に、出願や登録なしに**自動的に付与**される権利。

### 著作権の対象とならないもの

- **アルゴリズム**: 解法そのものは著作物として保護されない
- **プログラミング言語**: 文法やキーワード自体は保護されない（必要なら**特許**で保護）

### 2種類の著作権

| 種類 | 内容 | 譲渡・相続 |
|---|---|---|
| **著作者人格権** | 公表権、氏名表示権、同一性保持権など、著作者本人に固有の権利 | **不可** |
| **著作財産権** | 複製権、上演権、公衆送信権など、財産的な利用に関する権利 | 可能 |

## 産業財産権

特許庁への届出（出願・登録）が必要な権利。

| 権利 | 保護対象 | 例 |
|---|---|---|
| **特許権** | 高度な発明（自然法則を利用した技術的アイデア） | 新しい機械の構造、製造プロセス |
| **実用新案権** | 物品の形状・構造に関する小発明 | 既存品の改良アイデア |
| **意匠権** | 物品のデザイン | 製品の外観、形状 |
| **商標権** | 商品やサービスを識別するマーク | ロゴ、ブランド名 |

## セキュリティ関連法

| 法律 | 概要 |
|---|---|
| **サイバーセキュリティ基本法** | 国・地方公共団体・教育機関・事業者・国民の責務を定める基本法 |
| **不正アクセス禁止法** | 他人のID・パスワードを無断で利用する行為や、無断で他人に教える行為を禁止 |
| **個人情報保護法** | 個人情報を取り扱う事業者に対し、安全な管理・利用を義務付ける |
| **マイナンバー法** | マイナンバー（個人番号）の利用範囲・取扱いを定める |
| **特定電子メールの送信の適正化等に関する法律** | 受信を許可した人にのみ送る「**オプトイン**」が原則 |
| **プロバイダ責任制限法** | インターネット上の権利侵害について、プロバイダの損害賠償責任に制限を設け、発信者情報の開示請求手続きを定める |

## 刑法（コンピュータ関連犯罪）

| 罪名 | 内容 |
|---|---|
| **不正指令電磁的記録に関する罪** | マルウェアを作成・提供・取得・保管・実行する行為 |
| **電子計算機使用詐欺罪** | コンピュータを使って財産的利益を不正に得る詐欺 |
| **電子計算機損壊等業務妨害罪** | コンピュータやデータを損壊して業務を妨害する行為 |

## 労働関連法規

### 労働基準法

労働契約や労働条件、職場環境、年少者・妊産婦の保護、災害補償など、**労働条件の最低基準**を定めた法律。

- **法定労働時間**: 1日8時間、1週間40時間（合計）

### 36協定（さぶろくきょうてい）

法定労働時間を超える残業をさせるために、労働者と使用者で結ぶ協定（労働基準法第36条に基づく）。

#### 残業時間の上限

| 区分 | 原則 | 特別条項（特例） |
|---|---|---|
| **1か月あたり** | 45時間 | 100時間未満（休日労働を含む単月上限）<br>※複数月平均は80時間以内 |
| **1年あたり** | 360時間 | 720時間以内 |
| **特例の使用回数** | — | 年6回まで（45時間超の月） |
