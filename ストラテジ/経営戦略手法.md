# 経営戦略手法

**シラバス大分類：19. 経営戦略マネジメント**

## SWOT分析

自社の内部環境と外部環境を「プラス要因」「マイナス要因」で分類し、戦略立案に活用するフレームワーク。

|  | プラス要因 | マイナス要因 |
|---|---|---|
| **内部環境** | **Strength**（強み）<br>例: 高い技術力、強いブランド | **Weakness**（弱み）<br>例: 資金不足、ノウハウ不足 |
| **外部環境** | **Opportunity**（機会）<br>例: 市場拡大、規制緩和 | **Threat**（脅威）<br>例: 競合の参入、景気後退 |

## 成長マトリクス（アンゾフのマトリクス）

製品と市場の組み合わせから企業の成長戦略を分類するフレームワーク。

|  | 既存製品 | 新製品 |
|---|---|---|
| **既存市場** | **市場浸透戦略**<br>シェア拡大、購入頻度向上 | **新製品開発戦略**<br>既存顧客に新製品を投入 |
| **新市場** | **新市場開拓戦略**<br>海外進出、新セグメント開拓 | **多角化戦略**<br>未経験の市場に新製品で挑む |

## プロダクトポートフォリオマネジメント（PPM）

事業や製品を「市場成長率」と「市場占有率」の2軸で分類し、経営資源の配分を決める手法。

|  | 市場成長率: 高 | 市場成長率: 低 |
|---|---|---|
| **市場占有率: 高** | **花形**（Star）<br>投資を続けて地位維持 | **金のなる木**（Cash Cow）<br>稼ぎを他事業に振り向ける |
| **市場占有率: 低** | **問題児**（Question Mark）<br>育成 or 撤退の判断 | **負け犬**（Dog）<br>撤退を検討 |

## 競争地位別戦略

業界内の競争地位に応じて取るべき戦略を分類した枠組み。

| 地位 | 特徴 | 基本戦略 |
|---|---|---|
| **リーダー** | 業界トップシェア | 全方位戦略、市場全体の拡大 |
| **チャレンジャー** | 2位以下の上位企業 | 差別化戦略、リーダーへの攻撃 |
| **フォロワー** | 中位企業 | リーダー・チャレンジャーの模倣で低コスト化 |
| **ニッチャー** | 特定セグメントに特化 | 集中戦略、専門特化 |

## プロダクトライフサイクル（PLC）

製品が市場に投入されてから撤退するまでの段階。

1. **導入期**: 売上は少なく、認知獲得が課題。利益はマイナスが普通
2. **成長期**: 売上が急速に伸び、競合が増える
3. **成熟期**: 売上の伸びが鈍化し、シェア争いが激化
4. **衰退期**: 売上・利益ともに減少。撤退の判断が必要

## 3C分析

経営環境を3つの視点で分析するフレームワーク。

- **Customer**（顧客・市場）
- **Competitor**（競合）
- **Company**（自社）

## RFM分析

顧客を3指標で評価し、新規顧客・優良顧客・休眠顧客などにセグメント化する手法。

- **Recency**（直近購入日）
- **Frequency**（購入頻度）
- **Monetary**（購入金額）

3指標すべてが高い顧客を**ロイヤル顧客**という。

## バランススコアカード（BSC）

財務面だけでは捉えきれない企業価値を、4つの視点でバランスをとって評価する手法。3C分析の「自社（Company）」を深掘りするツールでもある。

1. **財務の視点**: 売上、利益、ROEなど
2. **顧客の視点**: 顧客満足度、シェア
3. **内部プロセスの視点**: 業務効率、品質
4. **学習と成長の視点**: 従業員のスキル、組織文化

## PEST分析

マクロ外部環境を4つの視点で分析するフレームワーク。

- **Politics**（政治）: 法規制、税制、政権動向
- **Economy**（経済）: 景気、為替、金利
- **Society**（社会）: 人口動態、ライフスタイル、文化
- **Technology**（技術）: 技術革新、特許、研究開発動向

## ファイブフォース分析

業界の収益性に影響する5つの競争要因を分析するフレームワーク。

1. **新規参入業者の脅威**
2. **売り手（供給業者）の交渉力**
3. **買い手（顧客）の交渉力**
4. **代替品の脅威**
5. **業界内の競合**（既存競合の敵対関係）

## VRIO分析

経営資源の競争優位性を4つの視点で評価する手法。

| 観点 | 質問 |
|---|---|
| **Value**（価値） | 経済的価値があるか |
| **Rarity**（希少性） | 市場で希少か |
| **Inimitability**（模倣困難性） | 競合に模倣されにくいか |
| **Organization**（組織） | 経営資源を活かす組織体制があるか |

V → R → I → O の順で問い、Yesが多いほど持続的競争優位を築ける。

## バリューチェーン分析

事業活動を機能ごとに分解し、どの活動で付加価値を生み出しているかを分析する手法。**主活動**と**支援活動**に分かれる。

### 主活動（Primary Activities）

1. **購買物流**: 原材料の受入・保管
2. **製造**: 製品化
3. **出荷物流**: 製品の保管・配送
4. **販売・マーケティング**: 販売活動、広告
5. **サービス**: アフターサービス、保守

### 支援活動（Support Activities）

1. **全般管理**（インフラ）
2. **人事・労務管理**
3. **技術開発**
4. **調達活動**
