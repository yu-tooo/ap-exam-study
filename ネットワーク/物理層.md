# 物理層

**シラバス大分類：10. ネットワーク**

通信ケーブルなど、物理的な信号の伝送を担う層。データの変換と物理接続の維持が主な役割。

## 物理層の機器

| 機器 | 役割 |
|------|------|
| **リピータ** | 信号を増幅・整形して**遠くまで届くように**する。 |
| **ハブ** | 入ってきた信号を**複数のポートに分岐**して送る（リピータハブの場合は単純に全ポートにブロードキャスト）。 |
| **NIC**（Network Interface Card） | コンピュータをネットワークに接続するためのインターフェース機器。MACアドレスを持つ。 |

## ケーブルの種類

| 種類 | 特徴 |
|------|------|
| **ツイストペアケーブル** | 2本の導線を撚り合わせた構造。**低コスト**で短〜中距離向け。LAN配線で広く使われる。 |
| **同軸ケーブル** | 中心導体を絶縁体・編組シールドで覆った構造。**長距離**伝送に向く。 |
| **光ファイバ** | 光信号を伝える。**長距離・高速・大容量**に優れる。電磁ノイズの影響を受けない。 |

## 規格名称（イーサネット規格）

例として `100BASE-TX` を分解すると以下の意味を持つ。

```
100  BASE  - TX
 │    │     │
 │    │     └─ 伝送媒体（T = ツイストペア、F = 光ファイバ など）
 │    └─────── 伝送方式（BASE = ベースバンド、BROAD = ブロードバンド）
 └──────────── 伝送速度（100 = 100Mbps、1000 = 1Gbps、10G など）
```

| 部分 | 意味 |
|------|------|
| 数字（先頭） | 伝送速度（Mbpsまたは Gbps）。例: `100` = 100Mbps、`1000` = 1Gbps。 |
| `BASE` / `BROAD` | 伝送方式。**BASE**＝ベースバンド伝送（デジタル信号をそのまま）、**BROAD**＝ブロードバンド伝送（変調して周波数多重）。 |
| 末尾の文字 | 伝送媒体。**T**＝ツイストペア、**F**＝光ファイバ など。`TX` の `X` は規格バージョンを表す。 |

## 光ファイバの形式

| 形式 | 特徴 |
|------|------|
| **マルチモード**ファイバ | 複数の光経路（モード）を許容する。**安価で接続が容易**。中距離向け。 |
| **シングルモード**ファイバ | 単一の光経路のみ通す。**高価だが長距離・広帯域**通信が可能。基幹回線や大規模ネットワークで使われる。 |

## 通信と電力の統合

電力供給と通信を**同一ケーブル**で行う技術。

| 規格 | 概要 |
|------|------|
| **PLC**（Power Line Communication） | **電力線通信**。家庭・建物内の電力線（コンセント配線）を通信回線として利用する。 |
| **PoE**（Power over Ethernet） | **イーサネットケーブル経由で電力供給**する。LANケーブル1本でデータ通信と電源を兼ねるため、無線APやIPカメラの設置で広く使われる。 |

> **Q. PLCとPoEの違いは？**
>
> A. 利用するケーブルとユースケースが異なる。
>
> | 観点 | PLC | PoE |
> |------|-----|-----|
> | 使うケーブル | **電力線**（コンセント配線） | **LANケーブル**（イーサネット） |
> | 主目的 | 電力線で**通信**を行う（既存配線の活用） | LANケーブルで**電力供給**を行う（電源工事の省略） |
> | 主な用途 | 屋内ネットワーク補助、産業機器制御 | 無線アクセスポイント、IPカメラ、IP電話 |
>
> 「PLCは電力線で通信」「PoEはLANで給電」と覚えるとよい。
