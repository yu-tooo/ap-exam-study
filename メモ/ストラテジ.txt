利益 = 収益 - 費用

営業利益 = 売上総利益 - 販売費・一般管理費

経常利益 = 営業利益 - (営業外費用 - 営業外収益)

税引き前純利益 = 経常利益 - (特別損失 - 特別利益)

変動費率 = 変動費 / 売上

損益分岐点 販売量 = (固定費) / (売上高 - 変動費)
損益分岐点 売上高 = 損益分岐点 販売量 × 売上高

固定費 = 10万円、売上 = 1個当たり250円、変動費 = 1個当たり50円とすると、

損益分岐点 販売量 = 100000 / (250 - 50) = 500(個)
損益分岐点 売上高 = 500 × 250 = 125000(円)

損益分岐点 売上高 = 固定費 / (1 - 変動比率) 変動比率 = 50 / 250

財務に関する指標

自己資本比率 = 自己資本 / 資本（自分のお金で手に入れているか = 企業の財政の健全性）
流動比率 = 流動資産 / 流動負債 （すぐにお金が返せる状況か = 短期的な企業の返済能力）
固定比率 = 固定資産 / 自己資本（固定資産を手に入れるためにどれくらいの自己資本を手に入れたか = 企業の財政の健全性）
総資本回転率 = 売上 / 総資本（総資本（自己資本と他人資本）によってどれだけ効率的に売上たか = 資本の使い方がうまい）
限界利益率 = 限界利益（売上高 - 変動費）/ 売り明け高（売れれば売れるほどもうかるか = 売上高の欧化による利益の増加の大きさ）
安全余裕率 = （売上高 - 損益分岐点）/ 売上高 (売上高に示す黒字分の割合 = 赤字になるまでの余裕)

ROI = もたらされた利益 / 投下した資本
ROA = 当期純利益 / 資産
ROE = 当期純利益 / 自己資本

4. レイトマジョリティ：市場の成熟期
5. ラガート：製品が完全に普及しきってから買う

2. 開発

3. 事業化
--- ダーウィンの海 ---
