# 人工知能

**シラバス大分類：1. 基礎理論**

## AIのレベル分類

人工知能は実装の高度さに応じてレベル分けされる（一般的な分類）。

| レベル | 概要 | 例 |
|-------|------|----|
| **レベル1** | 単純な制御プログラム | 自動で温度を調整するエアコン、全自動洗濯機 |
| **レベル2** | 古典的な人工知能（**エキスパートシステム**） | Amazonのおすすめ機能、お掃除ロボット |
| **レベル3** | 機械学習（**ニューラルネットワーク・統計的学習**を含む） | 将棋ソフト、検索エンジン |
| **レベル4** | ディープラーニング | 自動車の自動運転 |

## 機械学習

データからパターンを学習し、予測・判断を行う技術。学習方法によって以下に分類される。

### 教師あり学習

**正解データ（ラベル付きデータ）**を与えて、入力と正解の対応関係を学習する。分類・回帰問題で使われる。

- 例: 過去の画像に「猫」「犬」のラベルを付けて学習し、新しい画像を分類する。
- **過学習**（オーバーフィッティング）: 学習データにフィットしすぎて、未知データに対する予測精度が落ちてしまう現象。

### 教師なし学習

**正解データを与えずに**、データそのものの構造（似ているもの同士のグループなど）を学習する。

- 主な用途: クラスタリング（グループ分け）、次元削減、異常検知。

### 強化学習

動物の学習を模倣し、行動に対して**報酬**を与えることで、報酬を最大化する行動を学習する。

- 例: ゲームAI（囲碁・将棋）、ロボット制御、自動運転。

## ニューラルネットワーク

人間の脳の神経回路を模倣した数理モデル。基本構造は**入力層**と**出力層**から成る。各ノード（ニューロン）は重み付きの結合で接続される。

```
[入力層]      [出力層]
  ●─────────●
  ●─────────●
  ●─────────●
```

## ディープラーニング（深層学習）

ニューラルネットワークに**隠れ層（中間層）を複数追加**することで、より複雑な特徴を抽出できるようにした手法。画像認識・音声認識・自然言語処理で大きな成果を上げている。

```
[入力層]    [隠れ層1]   [隠れ層2]   [出力層]
  ●───────●───────●───────●
  ●───────●───────●───────●
  ●───────●───────●───────●
```

### バックプロパゲーション（誤差逆伝搬法）

出力層で得られた**誤差**（予測と正解の差）を、出力層から入力層に向かって**逆方向に伝搬**させ、各層の重みを修正する学習方法。多層ネットワークの効率的な学習を可能にした。
