# サイバー攻撃

**シラバス大分類：11. セキュリティ**

## 不正ログイン系

### SQLインジェクション

入力フォームなどから**不正なSQLを実行**させる攻撃。データベースの内容を改ざん・盗取される。

### パスワードクラック

パスワードのハッシュ値や暗号化データを**解析して、他人のパスワードを見破る**攻撃。

#### 対策手法

| 対策 | 内容 |
|------|------|
| **ソルト**（Salt） | ハッシュ化する**前に**付与するランダムな文字列。同じパスワードでも別のハッシュ値になり、レインボーテーブル攻撃を防げる。**ユーザーごとに異なる**値を使う。 |
| **ペッパー**（Pepper） | DB**以外**の場所（環境変数や設定ファイル）に保存する**共通の秘密文字列**。ハッシュ化前にパスワードに付与する。DBが漏洩してもペッパーが分からなければ復元が困難。 |
| **ストレッチング** | ハッシュ化を**何度も繰り返す**ことで、計算コストを増やし総当たり攻撃を困難にする。 |

### セッションハイジャック

ログイン時の**セッションID**を不正に入手し、正規ユーザーになりすまして不正ログインする攻撃。

### IPスプーフィング

パケットの**送信元IPアドレスを偽装**し、正しい送信元であると誤認させる攻撃。

## Webアプリケーション系

### OSコマンドインジェクション

入力値経由で**OSのコマンドを不正に実行**させる攻撃。

### SEOポイズニング

検索エンジン最適化（SEO）の手法を悪用し、検索結果の**上位に不正なサイトを表示**させ、利用者をマルウェア配布サイトへ誘導する攻撃。

### ドライブバイダウンロード

**ブラウザの脆弱性を悪用**し、Webサイトを**閲覧しただけで**マルウェアがダウンロード・実行されてしまう攻撃。

### クリックジャッキング

正規のページに**透明な不正ボタン（透明な iframe など）を重ねて配置**し、ユーザーを騙してクリックさせる攻撃。

## 端末・システム侵入系

### ゼロデイ攻撃

**脆弱性が修正される前**（パッチが提供される前）に、その脆弱性を悪用して攻撃する手法。

### バックドア

不正侵入した際に、**再侵入用の裏口**をあらかじめ仕掛けておくこと、またはそのプログラム。

### ボット

**感染したコンピュータをネットワーク経由で外部から操作**できるようにするマルウェア。多数のボットを束ねた**ボットネット**は DDoS 攻撃などに使われる。

### キーロガー

利用者が入力した**キー操作を記録**し、パスワードや個人情報を盗み出すマルウェア／プログラム。

## 物理・観測系

### サイドチャネル攻撃

攻撃対象の機器（ICカード・電力メーターなど）を**観測**し、消費電力・電磁波・処理時間などの**副次情報**から機密情報を得る攻撃。

### テンペスト攻撃

**電磁波の漏洩**を観測して情報を盗み取る、サイドチャネル攻撃の一種。ディスプレイやキーボードから漏れる電磁波から表示内容や入力を再現できる場合がある。

## 偽情報・ソーシャル系

### ディープフェイク

AI技術を使って作られた**偽の映像や音声**で、人物を誤認させる攻撃。本物そっくりの偽動画によるなりすまし・詐欺などに悪用される。

## サイバーキルチェーン

サイバー攻撃のプロセスを**段階的に整理**したフレームワーク。Lockheed Martin社が提唱。各段階で攻撃を阻止する対策を考えるのに使える。

```
偵察 → 武器化 → デリバリ → エクスプロイト → インストール → C&C → 目的の実行
```

| 段階 | 内容 |
|------|------|
| **偵察**（Reconnaissance） | 標的の情報収集 |
| **武器化**（Weaponization） | マルウェアの作成 |
| **デリバリ**（Delivery） | マルウェアの送達（メール添付・Web配信など） |
| **エクスプロイト**（Exploitation） | 脆弱性の悪用・実行 |
| **インストール**（Installation） | 標的端末への定着 |
| **C&C**（Command and Control） | 外部の指令サーバとの通信確立 |
| **目的の実行**（Actions on Objectives） | 情報窃取・破壊などの最終目的の実行 |
