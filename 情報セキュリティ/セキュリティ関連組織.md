# セキュリティ関連組織

## CISO（Chief Information Security Officer）

組織における**情報セキュリティの最高責任者**。情報セキュリティに関する意思決定組織のトップで、以下を担う。

- 情報セキュリティポリシーのレビュー
- 情報セキュリティに関する管理
- 社員教育の方針策定

## CSIRT（Computer Security Incident Response Team）

組織内に設置され、発生した**インシデントに迅速に対応する**ための組織。

### ナショナルCSIRT

**国レベル**のCSIRT。日本のナショナルCSIRTは **JPCERT/CC**（JPCERT コーディネーションセンター）。

### インシデントハンドリング

異常を検知してから解決するまでの一連の対応。

```
検知 → トリアージ（優先順位付け）→ インシデントレスポンス（対応）
```

## JPCERT/CC（JPCERT コーディネーションセンター）

日本における**ナショナルCSIRT**。インシデント情報の収集・調整・公表を行う。

### JVN（Japan Vulnerability Notes）

JPCERT/CCと**IPA**が共同運営する、**脆弱性情報を提供するポータルサイト**。

## IPAセキュリティセンター

サイバー攻撃に関する**情報収集・分析**を行う、独立行政法人 IPA（情報処理推進機構）の組織。

## PSIRT（Product Security Incident Response Team）

組織内に設置され、**自社製品**に関するセキュリティ向上を担う。

> CSIRTがコンピュータセキュリティ全体に関するインシデント対応を担うのに対し、**PSIRTは自社の製品に特化**している点で対比される。

## CRYPTREC（クリプトレック、Cryptography Research and Evaluation Committees）

電子政府で使用される**暗号技術の安全性を評価・推奨**するプロジェクト。総務省・経済産業省・NICT・IPAが連携して運営し、電子政府推奨暗号リスト（CRYPTREC暗号リスト）を策定する。

## J-CSIP（Initiative for Cyber Security Information sharing Partnership of Japan）

**企業間でサイバー攻撃の情報を共有**するための組織。標的型攻撃の情報を業種横断で共有することで、被害拡大の防止を図る。

## J-CRAT（サイバーレスキュー隊、Cyber Rescue and Advice Team against targeted attack）

**標的型サイバー攻撃に対する相談**を受け付ける組織。被害の拡大防止と早期復旧を支援する。

## SOC（Security Operation Center）

組織の**外部に存在する**（または社内に設置される）セキュリティ監視センター。24時間365日体制でセキュリティ問題に対応する。

## 関連組織の整理

| 組織 | 役割 | 対象 |
|------|------|------|
| **CISO** | 情報セキュリティの最高責任者 | 組織全体 |
| **CSIRT** | インシデント対応 | 組織内のセキュリティ全般 |
| **PSIRT** | 自社製品のセキュリティ対応 | 製品 |
| **JPCERT/CC** | ナショナルCSIRT | 日本全体 |
| **IPAセキュリティセンター** | 情報収集・分析 | 国内のサイバー攻撃情報 |
| **JVN** | 脆弱性情報ポータル | 脆弱性情報 |
| **CRYPTREC** | 暗号技術の安全性評価 | 電子政府推奨暗号 |
| **J-CSIP** | 企業間情報共有 | 標的型攻撃情報 |
| **J-CRAT** | 標的型攻撃の相談窓口 | 被害組織の救援 |
| **SOC** | セキュリティ監視・対応 | 組織のシステム稼働監視 |
